# Modern Full-Stack Developer Portfolio

A responsive one-page portfolio built with Next.js, TypeScript, and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Personalize

Edit `app/page.tsx` to replace:

- `Alex Morgan` and the `AM` logo
- `hello@example.com`
- GitHub and LinkedIn URLs
- Project names, descriptions, and technologies
- Experience and project counts
- The photo placeholder

Update the page title and description in `app/layout.tsx`.
